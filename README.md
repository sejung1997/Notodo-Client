<h3>페이지 목록</h3>

  1. 메인페이지

 - 진행도 체크(프로그레스바)
 - 할 일 목록 조회
 - 일정<br/>
 
 2. 첼린지 페이지

- 팀플 목록 조회
    - 팀플 상세 페이지
        - 팀 댓글(채팅)
        - 신청 현황
        - 일정

3. 커뮤니티 페이지
- 모집중인 팀플 목록
    - 댓글(+ 푸쉬 알람) → 채팅
    - 신청

4. 챌린지 생성

5. 마이페이지(유저정보)
- 실행률
- 참여했던 프로젝트(유저 상세 페이지)
- 신청 대기 목록(마이페이지)
- 정보 수정
